Structure of HTML

Understanding the foundational building blocks of the web.

The Boilerplate

Every HTML document starts with a standard structure known as the "boilerplate". It tells the browser what kind of document to expect and sets up the fundamental areas for metadata and content.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Page Title</title>
</head>
<body>
    <h1>Hello World</h1>
</body>
</html>

1. The Document Type

<!DOCTYPE html> is the very first line of code. It is not technically an HTML tag, but an instruction to the web browser indicating that the page is written in modern HTML5.

2. The Root Element

<html> wraps all the content on the entire page. It is known as the "root" element. Adding the lang="en" attribute helps search engines and screen readers understand the language of the page.

3. The Head Section

<head> contains meta-information about the HTML document that isn't displayed directly on the screen. This includes character sets, the page title (which appears in the browser tab), and links to external CSS stylesheets.

4. The Body Section

<body> is the canvas of your website! It contains all the visible contents of an HTML document, such as headings, paragraphs, images, tables, lists, and forms. Everything the user interacts with goes right here.
