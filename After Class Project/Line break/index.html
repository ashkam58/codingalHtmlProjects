Line Breaks in HTML

Mastering the <br> tag for precise text formatting.

The <br> Tag

The <br> tag produces a line break in text (carriage-return). It is known as an "empty" tag or void element, meaning it does not contain content and doesn't require a closing tag.

Unlike standard paragraphs (<p>) which create block-level spacing and margins, a line break simply forces the following text to the next line tightly.

Use Case 1: Poetry & Lyrics

When formatting poetry or lyrics, you want the lines to break exactly where you place them without the large vertical gaps generated by paragraph tags.

Roses are red,
Violets are blue,
HTML is awesome,
And so are you!

Code: Roses are red,<br>Violets are blue...

Use Case 2: Mailing Addresses

Physical addresses are another perfect scenario where line breaks should be used to keep the lines grouped tightly together.

John Doe
123 Developer Way
Tech City, TX 75001
United States

Code: John Doe<br>123 Developer Way<br>...
